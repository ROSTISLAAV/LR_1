1. git має 3 основних стани, в яких можуть перебувати файли: збережений у комміті(commited), змінений(modified) та індексований(staged).
2. git add - команда для додавання файла до репозиторію.
3. git status - перевіряє чи додані файли до репозиторію, зелений текст - так, червоний - ні.
4. git commit --m "Назва комміту" - фіксація стану сховища в певний момент часу.
5. git init - ініціалізація Git у папці проєкту.
